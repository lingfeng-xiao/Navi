## 组件状态

### 常规状态
- **默认状态**: 组件的初始显示状态
- **悬停状态**: 用户鼠标悬停时的状态变化
- **激活状态**: 组件被选中或激活时的状态

### 交互状态
- **加载状态**: 数据加载过程中的显示状态
- **错误状态**: 操作失败或数据错误时的显示状态
- **禁用状态**: 组件不可交互时的状态
