# 个人经济管理系统 - 设计文档

## 项目概述
本文档提供个人经济管理系统的完整设计规范，包括技术栈、项目结构、核心功能模块和UI/UX设计标准，为前端开发团队提供统一的设计参考。

## 技术栈

### 前端技术
- Vue 3 + TypeScript + Vite
- Element Plus：UI组件库
- ECharts：图表可视化
- Axios：HTTP请求
- Pinia：状态管理
- Vue Router：路由管理
- CSS Variables：样式变量管理

### 后端技术
- Python FastAPI
- MySQL
- Redis

## 核心功能模块

### 1. 数据大盘

#### 功能描述
提供个人财务总体概览，包括收支总额、预算执行情况、财务趋势等关键指标。

#### 布局架构
- **固定顶部导航**：包含logo、主要导航菜单、搜索、通知和用户信息
- **侧边栏导航**：可折叠，包含主要功能模块入口
- **主内容区**：
  - **顶部概览区**：4个关键指标卡片，带有动态数值变化动画
  - **中部分析区**：左右两栏布局，左侧为收入/支出趋势图，右侧为支出分类饼图
  - **预算跟踪区**：进度条卡片，显示各分类预算使用情况
  - **最近交易区**：交易记录表格，支持实时筛选和排序
- **浮动操作按钮**：快速添加交易记录

#### 响应式布局
- **桌面端(≥1200px)**：完整多列布局，所有卡片同时显示
- **平板端(768-1199px)**：双列布局，图表垂直排列
- **移动端(<768px)**：单列布局，卡片堆叠，侧边栏转为抽屉式菜单

#### 视觉层次
- **Z轴分层**：卡片悬浮在背景之上，导航栏位于顶层
- **信息优先级**：最重要的财务概览位于页面顶部，详细数据在下方
- **色彩区分**：使用主题色和功能色强调不同类型的数据

#### 交互功能
- **图表交互**：支持多维度数据筛选，图表数据动态更新动画
- **卡片交互**：卡片悬停效果，点击展开详情模态框
- **实时更新**：数据变化时的平滑过渡动画
- **时间选择器**：直观的日历式时间范围选择，支持快速预设选项
- **滚动体验**：平滑滚动，元素随滚动渐入

#### 数据字段
- 总收入：Number
- 总支出：Number
- 总余额：Number
- 预算执行率：Number
- 交易记录列表：Array<{id, amount, type, category, date, description}>

### 2. 收入管理

#### 布局架构
- **顶部操作栏**：搜索框、筛选器、排序选项、批量操作按钮
- **主要内容区**：
  - **收入概览卡片**：展示选定时间范围的总收入、平均收入、收入分类占比
  - **收入记录列表**：分页表格，包含状态指示器和快速操作
  - **收入趋势图**：位于表格上方，直观展示收入变化趋势
- **右侧快捷面板**：
  - **快速添加表单**：简化版收入录入表单
  - **最近编辑**：显示最近修改的记录
  - **收入分类统计**：小尺寸的分类占比图
- **浮动操作按钮**：快速添加新收入记录

#### 交互优化
- **批量操作**：支持多选记录进行删除、修改分类等操作
- **拖拽排序**：支持自定义列表显示顺序
- **实时过滤**：输入即筛选，无需点击确认
- **快速编辑**：表格内直接编辑部分字段
- **上下文菜单**：右键点击记录弹出操作菜单

### 3. 支出管理

#### 布局架构
- **顶部操作栏**：搜索框、多维度筛选器、排序选项、批量操作
- **主要内容区**：
  - **支出概览卡片**：展示选定时间范围的总支出、平均支出、预算使用情况
  - **支出记录列表**：分页表格，支持行内编辑和展开详情
  - **支出热力图**：以日历形式展示每日支出情况
- **右侧分析面板**：
  - **支出分类占比**：环形图展示各类别支出比例
  - **预算跟踪器**：各分类预算使用进度
  - **异常支出提醒**：标记超出预期的支出
- **浮动操作按钮**：快速添加新支出记录

#### 交互优化
- **智能分类建议**：基于历史记录自动推荐支出分类
- **批量编辑**：支持同时修改多条记录的相同字段
- **标签过滤**：支持多标签组合筛选
- **收支对比视图**：一键切换到与收入的对比视图
- **导出功能**：支持多种格式导出支出数据

### 4. 负债管理

#### 1. 布局架构
- **顶部概览区**：
  - **总负债卡片**：展示总负债金额、已还金额、剩余负债、下月还款额
  - **负债健康指标**：可视化展示负债比例、信用评分等关键指标
- **主要内容区**：
  - **负债记录列表**：采用卡片式布局展示每条负债，包含状态指示器
  - **还款计划日历**：直观展示未来还款日期和金额
  - **负债趋势图**：展示负债随时间变化的趋势
- **右侧分析面板**：
  - **负债分类统计**：不同类型负债的比例分析
  - **还款策略建议**：基于当前负债情况提供智能还款建议
  - **提前还款计算器**：模拟提前还款对总利息的影响
- **底部操作区**：批量操作按钮、导出报告按钮

#### 2. 交互优化
- **负债详情模态框**：点击负债卡片展开详细信息和还款历史
- **交互式还款计划**：可调整还款日期和金额，实时计算影响
- **拖拽排序**：可自定义负债列表显示顺序
- **状态变更动画**：负债状态变化时的过渡动画
- **智能提醒**：到期还款前的视觉提醒和通知
- **情境菜单**：右键点击负债记录弹出快捷操作菜单
- **数据导出**：支持多种格式导出负债报表

#### 3. 数据可视化
- **负债饼图**：展示不同类型负债占比
- **还款进度条**：直观显示每条负债的还款进度
- **利息累计图**：展示利息随时间的累计情况
- **现金流预测**：基于还款计划的未来现金流预测

## UI/UX设计规范

### 色彩系统

#### CSS变量管理
- 使用CSS变量（Custom Properties）进行主题管理
- 所有设计令牌集中在`src/styles/variables.css`中定义
- 通过`:root`选择器全局应用变量

#### 颜色变量定义

##### 主色调
- 主背景：#FAFBFC (`--color-bg-main`)
- 卡片背景：#FFFFFF (`--color-bg-card`)
- 悬停背景：#F5F7F9 (`--color-bg-hover`)

##### 边框色
- 浅色边框：#E8EDF2 (`--color-border-light`)
- 默认边框：#E2E8F0 (`--color-border-default`)
- 深色边框：#D4DCE6 (`--color-border-dark`)

##### 功能色
- 收入：#6BA8A3 (`--color-income`)
- 支出：#B8A9C9 (`--color-expense`)
- 负债：#8FA8B8 (`--color-debt`)
- 强调色：#7A9BA8 (`--color-accent`)

##### 状态色
- 成功：#7FB8A1 (`--color-success`)
- 警告：#B8A289 (`--color-warning`)
- 信息：#8FA8B8 (`--color-info`)
- 错误：#C89FA3 (`--color-error`)

##### 文字色
- 主文字：#4A5568 (`--color-text-primary`)
- 次文字：#718096 (`--color-text-secondary`)
- 第三级文字：#A0AEC0 (`--color-text-tertiary`)
- 禁用文字：#CBD5E0 (`--color-text-disabled`)

### 视觉风格

#### 1. 布局原则
- **分层设计**：采用Z轴分层，通过阴影、透明度和间距创建深度感
- **不对称平衡**：使用非对称布局，营造动感和现代感
- **卡片式设计**：主要内容封装在卡片中，卡片间有足够间距（20-24px）
- **响应式网格**：基于12列网格系统，确保布局一致性
- **动态空间分配**：根据内容重要性动态调整空间分配

#### 2. 形状与线条
- **圆角系统**：卡片 16px，按钮 10px，输入框 12px，小元素 8px
- **霓虹边框**：重要元素添加发光边框效果
- **几何形状**：使用几何图形作为背景装饰元素
- **动态曲线**：使用平滑的贝塞尔曲线作为装饰线条
- **玻璃拟态（Glassmorphism）**：半透明效果，带有模糊背景

#### 3. 光效与阴影
- **渐变背景**：主色渐变，如#00C6FF到#0072FF
- **光晕效果**：按钮和关键元素的发光效果
- **深度阴影**：多层次阴影，增强立体感
- **动态光效**：交互时的光照变化效果
- **发光文字**：标题文字的轻微发光效果

#### 4. 字体系统
- **数字/代码**：JetBrains Mono（科技感字体）
- **中文**：PingFang SC（首选）/ Microsoft YaHei（备选）
- **英文**：Inter（首选）/ SF Pro Display（备选）
- **字体层级**：
  - 大标题：28-36px，字重700
  - 中标题：20-24px，字重600
  - 小标题：16-18px，字重600
  - 正文：14-16px，字重400
  - 辅助文字：12-13px，字重400
- **行高**：正文1.5-1.6，标题1.2-1.3

#### 5. 交互设计
- **微动画**：所有可交互元素都有细微的动画反馈
- **过渡时长**：
  - 快速过渡：200-250ms（hover效果）
  - 中等过渡：300-400ms（页面元素切换）
  - 复杂过渡：500-600ms（页面加载）
- **缓动函数**：cubic-bezier(0.4, 0, 0.2, 1) 或 cubic-bezier(0.68, -0.55, 0.265, 1.55)
- **悬停效果**：
  - 按钮：轻微上浮（translateY(-3px)），发光效果增强
  - 卡片：轻微上浮（translateY(-4px)），阴影加深
  - 链接：颜色变化，下划线动画
- **点击效果**：轻微缩放（scale(0.97-0.98)）
- **数据加载**：骨架屏，平滑过渡到实际数据
- **页面切换**：平滑的转场动画，避免硬切
- **滚动效果**：滚动时元素的渐入效果，固定导航栏样式变化

##### 设计系统管理

### 设计令牌管理
- 使用CSS变量（Custom Properties）定义所有设计令牌
- 在`src/styles/variables.css`中集中管理
- 变量命名遵循`--[category]-[element]-[variant]`模式
- 所有组件应通过CSS变量引用设计值，而非硬编码

### CSS变量扩展
- **字体变量**：定义字体族、字重和字号
- **间距变量**：统一的间距系统
- **圆角变量**：不同级别的圆角值
- **阴影变量**：多层次阴影效果
- **渐变变量**：预设的渐变效果
- **过渡变量**：统一的过渡时长和缓动函数
- **Z-index变量**：明确的层级管理

### 组件变体和状态
- 为每个组件定义默认、悬停、聚焦、禁用等状态
- 提供不同尺寸的组件变体（小型、中型、大型）

### 响应式设计
- 移动端优先设计理念
- 断点设置：
  - 小屏手机：< 576px
  - 大屏手机/平板：576px - 768px
  - 小型桌面：768px - 992px
  - 中型桌面：992px - 1200px
  - 大型桌面：> 1200px

### 无障碍设计
- 所有交互元素支持键盘导航
- 重要内容提供足够的颜色对比度
- 图表提供替代文本说明
- 表单元素包含清晰的标签和错误提示
